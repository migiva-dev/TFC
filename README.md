# 💈 Peluquería TGC

Proyecto de fin de grado (DAW) — Aplicación web para gestión de reservas de peluquería.

## Tecnologías usadas
- PHP (backend)
- MySQL / phpMyAdmin (base de datos)
- HTML5 + CSS3 + JavaScript (frontend)
- Hospedado en AwardSpace

## Funcionalidades
- Web pública con información de servicios
- Sistema de registro e inicio de sesión para clientes
- Reserva de citas online
- Panel de administración privado con gestión de reservas
